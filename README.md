# token-inspector-ts

Addresses the provider to obtain a ERC20 **symbol**, **decimals**  
Realisation on TypeScript
# Layer Supertype

## Quick Start
1. `npm run build`
2. `npm run start`

## Expected output:  
```


                Token address: 0xdAC17F958D2ee523a2206206994597C13D831ec7
                Symbol: USDT
                Decimals: 6


```
